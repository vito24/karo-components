## ✨ 特性

- 开箱即用的高质量 React 组件。

## 📦 安装

```bash
npm install @karo/components --save
```

```bash
yarn add @karo/components
```

## 🔨 示例

```jsx
import { Button } from '@karo/components';
ReactDOM.render(<Button />, mountNode);
```
